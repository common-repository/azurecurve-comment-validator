== Description ==

Checks comment to ensure they are longer than the minimum, shorter than the maximum and also allows comments to be forced into moderation based on length.

This plugin is multisite compatible; each site can be set to use network settings or locally defined ones.

This plugin supports language translations. If you want to translate this plugin please sent the .po and .mo files to wordpress.translations@azurecurve.co.uk for inclusion in the next version (full credit will be given). The .pot fie is in the languages folder of the plugin and can also be downloaded from the plugin page on http://development.azurecurve.co.uk.

== Installation ==
To install the plugin copy the <em>azurcurve-comment-validator</em> folder into your plug-in directory and activate it.
